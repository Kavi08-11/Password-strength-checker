# Password Strength Checker - Python

This project is a simple Python program that evaluates the strength of a password entered by the user.  


# Features
  Checks:
  - Password length
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Displays level: 'Weak', 'Medium', or 'Strong'


# Technologies Used
- Python 3
- Regular Expressions (RegEx)


# How to Run

1. Install Python 
2. Download the project file: password_strength_checker.py
3. Open terminal or command prompt and run:



4. Enter a password when prompted.
